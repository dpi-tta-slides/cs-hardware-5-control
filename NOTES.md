## Lab Breakouts

### Lab #1: Name Badge

Participants create a scrolling name badge using the micro:bit LED display.

Concepts:
- Program upload
- LED matrix
- Output devices

### Lab #2: Digital Dice

Participants create a shake-to-roll digital dice.

Concepts:
- Accelerometer sensor
- Random numbers
- Event-driven programming

### Lab #3: Reaction Timer

Participants build a reaction-speed game.

Concepts:
- Inputs
- Outputs
- Timing
- Variables

### Lab #4: Rock Paper Scissors

Participants create a multiplayer game using radio communication.

Concepts:
- Wireless communication
- Events
- State machines

## Key Takeaways

- A microcontroller is a complete computer on a chip.
- Microcontrollers combine many concepts from previous lessons.
- Programs control hardware through inputs and outputs.
- Sensors allow computers to interact with the physical world.
- Embedded computers are everywhere.
- Physical computing connects software to real-world devices.
